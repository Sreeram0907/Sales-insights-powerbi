# Sales Insights Dashboard (SQL + Python + Power BI)

This project analyzes retail sales data using SQL and Python and presents
key business insights through an interactive Power BI dashboard.

The focus of the project is on transforming raw sales data into
decision-ready insights for business stakeholders.

---

## Project Overview

The project demonstrates an end-to-end analytics workflow:

1. Raw sales data is processed using Python
2. Structured data is stored in a SQL database
3. Power BI connects directly to the database
4. Interactive dashboards surface key performance metrics

---

## Tech Stack

- **Python** – data cleaning and transformation
- **SQL (MySQL)** – structured storage and aggregation
- **Power BI** – interactive dashboards and visualization

---

## Key Metrics & Insights

- Revenue trends over time
- Regional sales performance
- Top products by revenue and quantity
- Average order value (AOV)

---

## Dashboard Use Cases

- Identify high-performing regions and products
- Track monthly and seasonal sales trends
- Support data-driven business and product decisions

---

## Notes

This project adapts an open-source SQL analytics pipeline and extends it
with MySQL and Power BI to focus on interactive business reporting and
decision-making.
